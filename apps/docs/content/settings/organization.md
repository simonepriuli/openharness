Open **Settings → Organization** to manage your team workspace.

## Tabs

| Tab | Purpose | Access |
|-----|---------|--------|
| **Details** | Organization name and info | All members |
| **Members** | View and manage members | All members |
| **Source control** | GitHub and Azure DevOps connections | All members |
| **Integrations** | Teams and Discord | All members |
| **Runners** | Local runner bindings | All members |
| **Secrets** | API keys and service credentials | Admins only |

## Details

View and edit the organization name. Admins can update organization settings.

## Members

- View all organization members
- Admins can invite new members (invite codes)
- Admins can remove members

## Admin vs member

Admins see the **Secrets** tab and can manage organization-wide API keys. Members use shared secrets but cannot edit them.
