Open **Settings → Usage** to review how many tokens you have consumed across agent sessions.

## Totals

- **Total tokens** — cumulative tokens across all sessions on this machine
- **Tokens this month** — tokens used in the current calendar month

Counts are tracked locally in the desktop app and update as you chat with the agent.

## Activity heatmap

The heatmap shows daily token usage over the last 12 months:

- A date range label shows the period covered (e.g. Jul 2025 – Jul 2026)
- Each square is one day — darker green means more tokens
- Hover a square to see the exact count and date
- History accumulates from when this version starts recording daily usage forward

## Derived stats

Below the heatmap:

- **Most active month** — calendar month with the highest token total
- **Most active day** — single day with the highest token total
- **Longest streak** — longest run of consecutive days with usage
- **Current streak** — consecutive days with usage ending today (resets if today has no usage)

## Quick access

The workspace menu sidebar also shows total and monthly token counts. Click **View details** there to open this page.
