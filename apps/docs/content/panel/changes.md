---
title: Changes panel
description: View unstaged git diffs in the right panel.
---

The **Changes** tab shows a unified diff of your working tree — all unstaged modifications in the project.

## Diff view

- Added lines appear with `+` prefix
- Removed lines appear with `−` prefix
- Context lines help you locate changes in the file

## Empty state

When there are no unstaged changes, the panel shows an empty state message.

## Use cases

- Review agent edits before committing
- Verify the scope of changes in a conversation
- Cross-reference with [timeline file edit summaries](/chat/timeline)

## Header stats

The chat header also shows aggregate git stats (+added / −removed lines, changed file count) when your project is a git repository.
