Open **Settings → General** for application-wide preferences.

## Work mode

| Mode | Purpose |
|------|---------|
| **For coding** | Software development — repositories, Plan/Swarm, workflows, git panel, and office documents |
| **For everyday work** | Simplified layout for writing, research, and document workflows (same tools, fewer coding UI elements) |

**For coding** is the default. Switch modes here if you need the everyday work experience.

## Appearance

Choose a theme:

- **System** — Follow OS light/dark preference
- **Light** — Light interface
- **Dark** — Dark interface

You can also switch theme quickly from the sidebar footer.

## Updates

- **Check for updates** — Query for a new app version
- When an update is downloaded, an **Install** button appears in the chat header

On macOS, updates may also be available from the application menu. On other platforms, use Help → Check for Updates.

## New models notice

When an app update introduces new models, a dismissible dialog may appear prompting you to review available models.
