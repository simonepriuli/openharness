The left sidebar is your home base for switching between workflows, projects, and conversations.

## Sections

### Workflows

The **Workflows** entry at the top opens the workflows workspace. When a workflow run is active, a live indicator appears on this item.

### Repositories

Below Workflows, **Repositories** lists your open project folders. Each project expands to show its conversations.

## Sidebar header actions

| Action | How |
|--------|-----|
| Toggle sidebar | `Cmd/Ctrl+B` or header button when collapsed |
| Open folder | **+** button or `Cmd/Ctrl+O` |
| New conversation | **+** next to a project name or `Cmd/Ctrl+N` |

## Project row menu

Click **⋯** on a project row to access:

- **Connect GitHub repository** / **Disconnect** — Link local folder to a remote repo
- **Archive all chats** — Archive every conversation in the project
- **Remove project** — Remove from sidebar (does not delete files on disk)

## Conversation list

- Click a conversation to open it
- **Archive** individual chats from the conversation menu
- **Show more** appears after 5 threads
- A streaming indicator shows on conversations with active agent runs

## Sidebar footer

The footer panel provides quick links:

| Item | Opens |
|------|-------|
| Open folder | Folder picker |
| Settings | Settings overlay |
| Spending | OpenRouter credits (when configured) |
| Usage | Token usage summary |
| Theme | System / Light / Dark |
| Log out | Sign out |
