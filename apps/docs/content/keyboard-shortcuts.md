title: Keyboard shortcuts
description: Keyboard shortcuts for chat, navigation, and composer modes.
---

OpenHarness supports keyboard shortcuts for common actions. Shortcuts use `Cmd` on macOS and `Ctrl` on Windows/Linux.

## Global

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl+,` | Open Settings |
| `Cmd/Ctrl+O` | Open project folder |
| `Cmd/Ctrl+N` | New conversation (selected project) |
| `Cmd/Ctrl+B` | Toggle sidebar |

## Composer

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | New line in message |
| `Shift+Tab` | Cycle composer mode (Normal → Plan → Swarm) |
| `Cmd/Ctrl+Shift+S` | Toggle Swarm mode |
| `@` | Open file mention picker |
| `/` | Open slash command picker |

## Question panel

When the agent asks multiple-choice questions:

| Shortcut | Action |
|----------|--------|
| `a`–`z` | Select option by letter |
| `↑` / `↓` | Navigate options |
| `Enter` | Confirm selection |

## Application menu

Additional actions are available from the menu bar (macOS) or application menu (Windows/Linux):

| Menu | Item | Shortcut |
|------|------|----------|
| File | Settings | `Cmd/Ctrl+,` |
| File | Open Project | `Cmd/Ctrl+O` |
| File | New Conversation | `Cmd/Ctrl+N` |
| View | Toggle Sidebar | `Cmd/Ctrl+B` |
| View | Toggle Swarm Mode | `Cmd/Ctrl+Shift+S` |

## Search documentation

On this docs site, press `Cmd/Ctrl+K` to search all pages.
