In **For coding** mode, when the agent needs your input it can show a structured question panel above the composer.

## Question panel

The panel appears when the agent uses `ask_question` with multiple-choice options.

Each question shows:

- Question text
- Lettered options (a, b, c, …)
- Navigation: Previous, Skip, Next

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| `a`–`z` | Select option by letter |
| `↑` / `↓` | Navigate between options |
| `Enter` | Confirm selection |

## Batched questions

The agent may ask several questions in sequence. Use **Next** to advance and **Previous** to go back. **Skip** moves past a question without answering.

## Plan mode

The question panel is heavily used in [Plan mode](/chat/plan-mode) during the interview phase.

## Everyday work mode

In **For everyday work** mode, the question panel is disabled — the agent asks questions in plain text instead.
