The model switcher in the composer footer lets you choose which AI model powers the current conversation.

## Opening the model switcher

Click the model name in the composer footer to open the dropdown.

## Selecting a model

- Browse pinned models at the top
- Search all available models in the search field
- Click a model to switch immediately

Model changes apply to the current conversation session.

## Pinned models

Configure which models appear first in **Settings → Chat**. You can pin multiple models for quick access.

Click **Add models** in the switcher to open Chat settings when you need more options.

## Context gauge

When the session is ready, the composer shows:

- **Token usage** — Percentage of the model's context window used
- **Session cost** — Estimated cost for the current session (when available)

High context usage may require starting a new conversation for very long threads.

## Available models

Models come from your configured providers:

| Source | Configuration |
|--------|---------------|
| Organization secrets | Settings → Organization → Secrets |
| OAuth providers | Settings → OAuth providers |
| Local providers | Settings → Local providers |
