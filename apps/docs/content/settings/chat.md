Open **Settings → Chat** to configure chat-related preferences.

## Pinned models

Select which models appear in the composer model switcher. Pin your most-used models for quick access.

- Add models from all available providers
- Remove models you no longer need
- Order determines display in the switcher dropdown

## Title generation model

Choose the model used to auto-generate conversation titles from your first message.

A smaller, faster model is usually sufficient for titles.
