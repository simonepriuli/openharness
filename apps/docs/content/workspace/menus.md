OpenHarness provides desktop menus and a system tray for quick access to common actions.

## File menu

| Item | Shortcut | Action |
|------|----------|--------|
| Settings | `Cmd/Ctrl+,` | Open settings overlay |
| Open Project | `Cmd/Ctrl+O` | Open a folder |
| New Conversation | `Cmd/Ctrl+N` | Start a new thread |
| Quit / Close | — | Exit the application |

On macOS, Settings is under the application menu instead of File.

## View menu

| Item | Shortcut | Action |
|------|----------|--------|
| Toggle Sidebar | `Cmd/Ctrl+B` | Show or hide sidebar |
| Toggle Swarm Mode | `Cmd/Ctrl+Shift+S` | Enable or disable Swarm mode |
| Appearance | — | System, Light, or Dark theme |
| Zoom / Fullscreen | — | Standard zoom controls |

## Help menu

| Item | Action |
|------|--------|
| OpenHarness Documentation | Opens this documentation site |
| Pi Agent Documentation | Opens Pi agent docs |
| Check for Updates | Check for app updates (non-macOS) |
| View Release Notes | GitHub releases page |
| Report an Issue | GitHub issue form |

## System tray (macOS)

When the app runs in the background, the menu bar icon provides:

- **Open** — Bring OpenHarness to front
- **Settings** — Open settings
- Workflow busy indicator when runs are active
- **Quit**
