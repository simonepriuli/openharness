The right panel sits beside the chat workspace and provides quick access to project files, git changes, and plan documents.

## Opening the panel

Click the panel toggle button in the chat header. The panel is resizable — drag the divider to adjust width.

## Tabs

| Tab | Purpose |
|-----|---------|
| **Files** | Browse and preview project files |
| **Changes** | View unstaged git diffs |
| **Plan** | Preview plan documents from Plan mode |

Tabs are only available in **For coding** mode.

## When to use each tab

- **Files** — Navigate the codebase, preview file contents, use [code actions](/panel/code-actions) on selections
- **Changes** — Review what the agent (or you) changed before committing
- **Plan** — Read and implement [Plan mode](/chat/plan-mode) documents
