The **Files** tab shows a tree of your project directory with git-aware decorations.

## File tree

- Expand and collapse folders
- Click a file to open a preview
- Git status decorations indicate modified, added, or untracked files

## File preview

Selected files open in a syntax-highlighted code preview. Scroll to read file contents without leaving the chat workspace.

## Code selection

Select text in the preview to show the [code actions toolbar](/panel/code-actions):

- Explain
- Bug discovery
- Refactor
- Add tests
- Document

Each action injects a prefilled prompt into the composer referencing the selection, file path, and line numbers.
